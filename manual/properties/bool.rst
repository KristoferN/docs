Bool
============

.. attribute:: type: bool

The property output a single checkbox, when clicked the output value will be true otherwise false.

Settings
-----------

No settings exists.

Example
-----------

.. code-block:: php

  <?php

  $this->property([
      'title'    => 'True or false',
      'slug'     => 'my_true_or_false_slug',
      'type'     => 'bool'
  ])

Example output
-----------

.. code-block:: php

  string 'Fredrik'
