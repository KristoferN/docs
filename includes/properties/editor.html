NAV

Editor

type editor

since 1.2.0

Description

This property is a shortcut property that extends property text and changes the default value of editor to true.

Settings

No settings exists.

Example

papi_property([
  'title' => 'Editor',
  'slug'  => 'my_editor_slug',
  'type'  => 'editor'
])

Example output

String of text.
